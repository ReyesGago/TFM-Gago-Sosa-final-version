} <!-- ODF styles with no properties representable as CSS --> .dp1 .dp2 .dp3 .dp4 .T1 .T47 { } “COR PULMONALE” DEFINIZIOACor Pulmonale: Eskuineko bentrikuluaren egitura edo funtzioaren alterazioa, birikako hipertentsio arterialak sortutakoa.Birikako hipertentsio arteriala, birikaren gaixotasun edo birika-zirkulazio gaixotasunen ondorioz izandaiteke. Ezkerreko bihotz disfuntzioen edo sortzetiko kardiopatien ondorioz bada, ez da Cor Pulmonaletzat hartzen.Cor Pulmonaletzat hartzen dira ere, kaxa torazikoaren gaixotasun murriztaileak edota aireztapenkontrolaren alterazioak. Cor pulmonalearen ezaugarria, birikako hipertentsio da ( hipertentsio pre-kapilarra ) COR PULMONALEABirikako hipertentsio arterialaren neurketa:•Eskuineko bihotz-kateterismoaren bidez ( batez besteko presio arteriala > 25 mmHg (8-20 mmHg). •Doppler ekokardiogramaren bidez ( presio sistolikoa >40 mmHg) (GT). COR PULMONALEACor pulmonalea, eragilearen arabera, akutua edo kronikoa izan daiteke.1)Cor Pulmonale akutua: (ESB dilatazioa/ Birika tronboenbolismoa) 2) Cor pulmonale kronikoa: (ESB hipertrofia (+) + dilatazioa / Biriketako gaixotasun buxatzaile kronikoa) ETIOLOGIA1) Biriketako gaixotasunak•Biriketako gaixotasun buxatzaile kronikoa (%80) •Fibrosi kistikoa •Biriketako gaixotasun interstiziala 2) Birika-zirkulazioaren alterazioak•Birika-tronboenbolismoa •Biriketako hipertentsio primarioa •Anemia faltziformea •Eskistosomiasia •Biriketako gaixotasun bena-buxatzailea 3) Kaxa torazikoaren deformitateak•Zifoeskoliosia 4) Gaixotasun neuromuskularrak•Esklerosi lateral amiotrofikoa •Miastenia larria •Poliomielitisa •Guillain-Barre sindromea •Orno-muineko gaixotasunak •Diafragmaren aldebiko paralisia 5) Aireztapen kontrolaren alterazioak•Hipobentilazio zentrala •Loaldiko apnea sindromea FISIOPATOLOGIACor pulmonalean, birikako hipertentsio arterialaren fisiopatologian, bi gertakari dira nagusi: 1) Birikako odol-hodien uzkurdura (Biriketako gaixotasun buxatzaile kronikoa) albeoloren hipoxia → hodi-uzkurdura itzulgarria → hipoxia iraunkorra → aldaketa organikoak odol-hodietan (erdiko geruzaren hipertrofia, barne-geruzaren fibrosia) → ↑ presioa2) Birikako odol- hodien murriztapen anatomikoa ( enfisema, birikako enbolia…) ↓ odol- hodien murriztapena ( % ≥50) → ↑odol- hodien erresistentzia → ↑presioa ADIERAZPEN KLINIKOAKBirikako hipertentsioaren adierazpen klinikoak ez dira izaten oso nabariak hipertentsioa larria izan arte, nabarmenagoak izaten dira birikako hipertentsioaren eragiten duen gaixotasunaren adierazpen klinikoak. Horregatik Cor pulmonale kronikoaren diagnostikoa berandu egiten da, eskuin bihotz gutxiegitasuna agertzen denean.Birikako hipertentsioaren adierazpen klinikoak •Esfortzuagatiko disnea •Nekea •Bihotz aurreko mina •Esfortzuagatiko sinkopea •Edema. • Anorexia eta eskuin- hipokondrioaren mina •Eztula , hemoptisia eta erlastasuna (Ortner-en sindromea: birika-arteria dilatatuta dagoenean, ezkerreko nerbio laringeo errekurrentea, zanpatu egiten duelako).
